常见问题
如何创建第一个祝福？
点击首页的"+"按钮或"创建祝福"卡片，选择祝福对象，输入祝福内容，选择祝福特效，最后点击发送即可。您可以选择模板或自定义祝福语。
如何添加和管理考生信息？
进入"考生管理"页面，点击"添加考生"按钮，填写考生姓名、学校、联系方式等信息。您可以为每个考生设置备注和关注级别。
金币如何获得和使用？
通过每日签到、创建祝福、邀请好友等方式获得金币。金币可用于购买祝福特效、个性化头像框、专属祝福模板等道具。
如何查看我的祝福统计数据？
在个人中心点击"数据统计"，可以查看您的祝福数量、获得点赞数、活跃天数等详细统计信息，还能看到月度和年度趋势图表。
忘记密码怎么办？
在登录页面点击"忘记密码"，输入注册时的手机号码或邮箱，系统会发送重置密码的验证码，按照提示重新设置密码即可。
